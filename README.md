# HR Analytics Power BI Dashboard

## Project Overview
This project presents an **HR Analytics Dashboard built using Microsoft Power BI** to analyze employee attrition and workforce trends. The dashboard helps organizations understand key factors influencing employee turnover and supports data-driven HR decision making.

The analysis is based on the **IBM HR Analytics Employee Attrition dataset**, which contains employee demographic information, job roles, income levels, job satisfaction scores, and attrition status.

---

## Objectives
The main objectives of this project are:

- Analyze employee attrition patterns
- Identify departments and job roles with higher turnover
- Understand the relationship between salary, job satisfaction, and attrition
- Visualize workforce data using interactive Power BI dashboards

---

## Tools and Technologies
- Microsoft Power BI Desktop – Data visualization and dashboard development
- Power Query – Data cleaning and transformation
- DAX (Data Analysis Expressions) – Calculations and measures

---

## Dataset
The dataset used for this project is the **IBM HR Analytics Employee Attrition Dataset**.

It includes employee attributes such as:

- Age
- Department
- Job Role
- Monthly Income
- Gender
- Job Satisfaction
- Attrition Status

Total Records: **1470 employees**

---

## Dashboard Features
The HR Analytics dashboard provides insights through several visualizations, including:

- Total Employees
- Employees Left
- Attrition Rate
- Average Monthly Income
- Attrition by Department
- Attrition by Job Role
- Attrition by Age Group
- Attrition by Income Group
- Attrition by Gender
- Job Satisfaction Analysis

These visualizations help identify important trends related to employee attrition.

---

## Key Insights
- The overall **attrition rate is approximately 16%**.
- Certain job roles such as **Laboratory Technician and Sales Executive show higher attrition**.
- The **Research & Development department has the highest employee count and attrition**.
- Employees in **lower income groups contribute to a large portion of attrition**.
